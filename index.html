Guess the 5-letter word within 6 attempts.

  • Input letters using either your device's keyboard or the onscreen keyboard.
  • To check if your guess is correct, hit enter after each word attempt.
  • If a box turns green, the letter is correct.
  • If a box turns dark grey, the letter is incorrect.
  • If a box turns orange, the letter is correct but not in the correct place.
  • After 6 attempts or a correct word guess, a message will show up onscreen to confirm success/failure.
